Oliver-Taleb Phase Shift

Euly Oliver, Loïc Taleb

Definition

\overset\alpha{\mathrm{P}}f(x) = {1/2\pi} \int_{-\infty}^\infty\!\hat f(\xi) e^{i\xi x}e^{-i\,\mathrm{sgn}\(\xi\)\alpha}\,d\xi
\hat f(\xi) =\int_{-\infty}^{\infty}\!f(t) e^{-i\xi t}\,dt
\mathrm{sgn}\(x\)=\begin{cases}-1&\text{if }x\lt0\\\ \,0&\text{if }x=0\\\ \,1&\text{if }x\gt0\end{cases}

Properties

Additivity: \overset\alpha{\mathrm{P}}({f+g})(x)=\overset\alpha{\mathrm{P}}f(x)+\overset\alpha{\mathrm{P}}g(x)
Proof: Euly
\begin{aligned} \overset\alpha{\mathrm{P}}({f+g})(x) &\overset{\ast^1}= {1/2\pi} \int_{-\infty}^\infty\!(\hat f\(\xi\)+\hat g\(\xi\))e^{i\xi x}e^{-i\,\mathrm{sgn}\(\xi\)\alpha}\,d\xi &= {1/2\pi} \int_{-\infty}^\infty\!\hat f\(\xi\)\,e^{i\xi x}e^{-i\,\mathrm{sgn}\(\xi\)\alpha}+\hat g\(\xi\)\ e^{i\xi x}e^{-i\,\mathrm{sgn}\(\xi\)\alpha}\,d\xi &= {1/2\pi} \int_{-\infty}^\infty\!\hat f\(\xi\)\,e^{i\xi x}e^{-i\,\mathrm{sgn}\(\xi\)\alpha}\,d\xi+{1/2\pi} \int_{-\infty}^\infty\!\hat g\(\xi\)\ e^{i\xi x}e^{-i\,\mathrm{sgn}\(\xi\)\alpha}\,d\xi &= \overset\alpha{\mathrm{P}}f(x)+\overset\alpha{\mathrm{P}}g(x) \end{aligned}
\ast^1\ \ Implied by linearity of Fourier transform.


Homogeneity: \overset\alpha{\mathrm{P}}c*\!f\(x\) = c*\overset\alpha{\mathrm{P}}f\(x\)
Proof: Euly
\begin{aligned} \overset\alpha{\mathrm{P}}c*\!f\(x\) &\overset{\ast^1}= {1/2\pi} \int_{-\infty}^\infty\!c*\!\hat f\(\xi\)\,e^{i\xi x}e^{-i\,\mathrm{sgn}\(\xi\)\alpha}\,d\xi &= c*{1/2\pi} \int_{-\infty}^\infty\!\hat f\(\xi\)\,e^{i\xi x}e^{-i\,\mathrm{sgn}\(\xi\)\alpha}\,d\xi &= c*\overset\alpha{\mathrm{P}}f\(x\) \end{aligned}
\ast^1\ \ Implied by linearity of Fourier transform.


Identity: \overset{\pi}{\mathrm{P}}f\(x)=-f\(x)
Proof: Euly
\begin{aligned} \overset\pi{\mathrm{P}}f\(x\) &= {1/2\pi} \int_{-\infty}^\infty\!\hat f\(\xi\)\,e^{i\xi x}e^{-i\,\mathrm{sgn}\(\xi\)\pi}\,d\xi &= {1/2\pi}(\int_{-\infty}^0\!\hat f\(\xi\)\,e^{i\xi x}e^{i\pi}\,d\xi + \int_0^\infty\!\hat f\(\xi\)\,e^{i\xi x}e^{-i\pi}\,d\xi) &= {1/2\pi}(\int_{-\infty}^0\!\hat f\(\xi\)\,e^{i\xi x}(-1)\,d\xi + \int_0^\infty\!\hat f\(\xi\)\,e^{i\xi x}(-1)\,d\xi) &= -{1/2\pi}(\int_{-\infty}^0\!\hat f\(\xi\)\,e^{i\xi x}\,d\xi + \int_0^\infty\!\hat f\(\xi\)\,e^{i\xi x}\,d\xi) &= -{1/2\pi}\int_{-\infty}^\infty\!\hat f\(\xi\)\,e^{i\xi x}\,d\xi &= -f(x) \end{aligned} \begin{aligned} \overset\alpha{\mathrm{P}}f\(x) &= {1/2\pi}\int_{\R}\hat f\(\xi\) \ e^{-i\,\mathrm{sgn}\(\xi\)\alpha} e^{i\xi x} \ d\xi &= {1/2\pi}\!(e^{i\alpha}\!\int_{-\infty}^0\hat f\(\xi\) \ e^{i\xi x} \ d\xi+e^{-i\alpha}\!\int_0^\infty\hat f\(\xi\) \ e^{i\xi x} \ d\xi) \overset{-\alpha}{\mathrm{P}}f\(x) &= {1/2\pi}\!(e^{-i\alpha}\!\int_{-\infty}^0\hat f\(\xi\) \ e^{i\xi x} \ d\xi+e^{i\alpha}\!\int_0^\infty\hat f\(\xi\) \ e^{i\xi x} \ d\xi) \end{aligned}


Identity: \overset{\alpha}{\mathrm{P}}f\(x)+\overset{-\alpha}{\mathrm{P}}f\(x)=2\cos(\alpha)f\(x)
Proof: Loïc
\begin{aligned} \overset\alpha{\mathrm{P}}f\(x) &= {1/2\pi}\int_{\R}\hat f\(\xi\) \ e^{-i\,\mathrm{sgn}\(\xi\)\alpha} e^{i\xi x} \ d\xi &= {1/2\pi}\!(e^{i\alpha}\!\int_{-\infty}^0\hat f\(\xi\) \ e^{i\xi x} \ d\xi+e^{-i\alpha}\!\int_0^\infty\hat f\(\xi\) \ e^{i\xi x} \ d\xi) \overset{-\alpha}{\mathrm{P}}f\(x) &= {1/2\pi}\!(e^{-i\alpha}\!\int_{-\infty}^0\hat f\(\xi\) \ e^{i\xi x} \ d\xi+e^{i\alpha}\!\int_0^\infty\hat f\(\xi\) \ e^{i\xi x} \ d\xi) \end{aligned} \begin{aligned} \overset{\alpha}{\mathrm{P}}f\(x)+\overset{-\alpha}{\mathrm{P}}f\(x) &= {1/2\pi}\!(e^{i\alpha}\!\int_{-\infty}^0\hat f\(\xi\) \ e^{i\xi x} \ d\xi+e^{-i\alpha}\!\int_0^\infty\hat f\(\xi\) \ e^{i\xi x} \ d\xi) & \ \ \ \ \ + {1/2\pi}\!(e^{-i\alpha}\!\int_{-\infty}^0\hat f\(\xi\) \ e^{i\xi x} \ d\xi+e^{i\alpha}\!\int_0^\infty\hat f\(\xi\) \ e^{i\xi x} \ d\xi) &= {1/2\pi}\!((e^{i\alpha}\!+\!e^{-i\alpha})\!\int_{-\infty}^0\hat f\(\xi\) \ e^{i\xi x} \ d\xi+(e^{i\alpha}\!+\!e^{-i\alpha})\!\int_0^\infty\hat f\(\xi\) \ e^{i\xi x} \ d\xi) &= {e^{i\alpha}+e^{-i\alpha}/2\pi}\!(\int_{-\infty}^0\hat f\(\xi\) \ e^{i\xi x} \ d\xi+\int_0^\infty\hat f\(\xi\) \ e^{i\xi x} \ d\xi) &= {\cos(\alpha)/\pi}\int_{\R}\hat f\(\xi\) \ e^{i\xi x} \ d\xi &= 2\cos(\alpha)f\(x) \end{aligned}
